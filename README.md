AlaroxFileManager v0.42
=================


Simple FileManager: allows to:
*   Create a file,
*   Load a file,
*   Check if a file exist,
*   Write into a file,
*   Move/rename a file,
*   Delete a file.

Able to specifically load :
*   PHP files (.php),
*   XML files (.xml),
*   YAML files (.yml, .yaml)

Else, load with default php function ```file_get_contents()```
